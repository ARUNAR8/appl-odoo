# Appl.exe

Appl.exe is a Human Resource Management System (HRMS) developed for the Odoo Hackathon. It streamlines HR operations by providing secure authentication, employee management, attendance tracking, leave management, payroll visibility, and role-based access control.

---

## 📁 Project Structure

```text
Appl.exe/
│
├── frontend/      # React frontend
├── backend/       # Node.js + Express backend
├── database/      # SQL scripts and database schema
│     └── hrms.sql
├── docs/          # Documentation and requirements
└── README.md      # Project overview
```

---

## 🚀 Tech Stack

### Frontend
- React
- React Router
- Axios
- Tailwind CSS

### Backend
- Node.js
- Express.js
- JWT Authentication
- bcrypt

### Database
- MySQL

## ✨ Features

- Attendance Tracking
